## Getting Started

# Install

`npm install`

# Development

`npm run initdb`
`npm run dev` run on port 8080

# Default user

<b>Username:</b> root
<b>Password:</b> root

# Production

`npm run build`
`npm run start`

# Test

`npm run test`
or
`npm run testheaded`

# Open DB editor

`npm run studio`

# Environment variables

`SITE_URL` domain of the site
`PORT` only on production mode
